# results

This folder holds outputs generated by the notebooks and
`src.postprocessing.save_result(...)`: CSVs of the final trade matrix
`X`, per-country summaries, and any HTML/PNG exports of plots.

It is **git-ignored** (except for the `.gitkeep` placeholder) so that
large generated files don't end up in version control.

Re-generate everything by running the notebooks in `../scripts/`.
